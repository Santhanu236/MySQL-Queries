# Basic Database Queries:
### Listing Databases:
``` syntax
SHOW DATABASES;
```
#### Output:
| Database           |
|:-----|
| information_schema |
| mysql              |
| performance_schema |
| san                |
| sys                |
***

### Creating a Database :
``` syntax
CREATE DATABASE;
```
***

### Deleting a Database:
``` syntax
DROP DATABASE *DATABASE_NAME*
```
***
